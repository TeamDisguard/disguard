# @disguard/ts-config

**Typescript configuration file for [disguard.org](https://disguard.org/).**

</div>

## Installation

```shell
# yarn
> yarn add -D @disguard/ts-config
# npm 
> npm i -D @disguard/ts-config
```

## Usage

Create a `tsconfig.json` file:
```json
{
  "extends": "@disguard/ts-config"
}
```
